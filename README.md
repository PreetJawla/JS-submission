--NFT Collection Manager--
This is a simple JavaScript program to manage a collection of NFTs (Non-Fungible Tokens). 
It allows you to mint new NFTs with metadata and list existing NFTs in the collection.

--Getting Started--
To use this program, you can follow these steps:

-Clone the repository to your local machine.
-Open the index.js file in a JavaScript environment (e.g., a web browser console, Node.js environment).
-Call the provided functions to mint new NFTs and list existing ones.

--Functions--

-mintNFT(fruit, color, taste, weight)
This function mints a new NFT with the specified metadata and adds it to the collection.

fruit: The type of fruit for the NFT.
color: The color of the fruit.
taste: The taste of the fruit.
weight: The weight of the fruit.

-listNFTs()
This function lists all the NFTs in the collection along with their metadata.

-getTotalSupply()
This function returns the total number of NFTs minted in the collection.
